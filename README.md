Machine learning models for time series forecasting, including traditional statistical methods and modern deep learning approaches.

## Overview

This repository provides implementations and comparisons of various machine learning techniques for time series forecasting. It covers data preprocessing, feature engineering, model training, and evaluation for predicting future values based on historical data.

## Features

-   **Data Preprocessing**: Handling missing values, stationarity tests, and differencing.
-   **Feature Engineering**: Lag features, rolling statistics, and seasonal components.
-   **Traditional ML Models**: ARIMA, Prophet, and other statistical models.
-   **Deep Learning Models**: LSTMs and GRUs for sequence prediction.
-   **Evaluation Metrics**: RMSE, MAE, and MAPE for model performance assessment.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Python 3.8+
-   pip

### Installation

1.  Clone the repo
    ```bash
    git clone https://github.com/Constainey5/Time-Series-Forecasting-ML.git
    ```
2.  Install Python packages
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Explore the `notebooks/` directory for detailed examples and the `src/` directory for reusable forecasting modules.

```bash
python src/forecast_example.py
```
